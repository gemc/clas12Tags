Based on [Coatjava Release](https://github.com/JeffersonLab/coatjava/releases) 13.8.3


# Major release

This major release supports real run numbers: it allows
running gemc with run number different than 11 and enable realistic 
constants during digitization.

The changes to the hit process routines, made by Cole,Raffaella,and Veronique  are shown below. 

# Release notes

- CTOF: added fadc-tdc offset and hpos correction
- DC: accounting for start time, tflight, and prop, switched to v2 CCDB tables including beta-dependent tie walk
- FTOF: added fadc-tdc offset, energy- and position-dependent time walk
- HTCC: using gain from CCDB
- added time jitter to all TDCs
- restores ECAL hitprocess modifications to match "pass2" reconstruction
- BMT: reading HV for Lorentz angle calculation from CCDB
